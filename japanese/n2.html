N2 文法

欄位：句型／接續／中文意思／日文例句／中文翻譯。

にあたって

接續辞書形／名詞 + にあたって

意思在～之際

新しい仕事を始めるにあたって、準備をしました。
開始新工作之際做了準備。

に際して

接續辞書形／名詞 + に際して

意思在～時

入国に際して、パスポートが必要です。
入境時需要護照。

に先立って

接續名詞 + に先立って

意思在～之前

会議に先立って資料を配りました。
會議前發了資料。

を問わず

接續名詞 + を問わず

意思不問～

年齢を問わず参加できます。
不分年齡都能參加。

にかかわらず

接續名詞／普通形 + にかかわらず

意思不管～

理由にかかわらず遅刻は困ります。
不管理由，遲到都很麻煩。

にもかかわらず

接續普通形 + にもかかわらず

意思儘管～卻～

雨にもかかわらず試合は行われました。
儘管下雨比賽仍舉行。

からすると

接續名詞 + からすると

意思從～來看

彼の表情からすると、失敗したようです。
從他的表情來看好像失敗了。

に基づいて

接續名詞 + に基づいて

意思根據～

データに基づいて判断します。
根據資料判斷。

に応じて

接續名詞 + に応じて

意思依照～

経験に応じて給料が変わります。
依經驗薪水不同。

に沿って

接續名詞 + に沿って

意思沿著；按照

計画に沿って進めます。
按照計畫推進。

を中心に

接續名詞 + を中心に

意思以～為中心

若者を中心に人気があります。
以年輕人為中心很受歡迎。

をめぐって

接續名詞 + をめぐって

意思圍繞～

新制度をめぐって議論が続いています。
圍繞新制度持續討論。

に反して

接續名詞 + に反して

意思違反；與～相反

予想に反して売れました。
與預測相反賣得很好。

に比べて

接續名詞 + に比べて

意思與～相比

去年に比べて暑いです。
和去年相比很熱。

一方で

接續普通形 + 一方で

意思另一方面

この町は便利な一方で、家賃が高いです。
這個城鎮方便，但另一方面房租高。

一方だ

接續辞書形 + 一方だ

意思一直～

物価は上がる一方です。
物價一直上漲。

ものの

接續普通形 + ものの

意思雖然～但是

資格は取ったものの、まだ自信がありません。
雖然取得資格，但還沒自信。

ことから

接續普通形 + ことから

意思由於～可知；因為

便利なことから人気があります。
因為方便所以受歡迎。

ことなく

接續辞書形 + ことなく

意思沒有～就～

休むことなく働き続けました。
沒有休息地持續工作。

ないことには

接續ない形 + ことには

意思如果不～就不能～

実際に見ないことには判断できません。
不實際看就無法判斷。

ざるを得ない

接續ない形去ない + ざるを得ない

意思不得不～

予定を変更せざるを得ません。
不得不更改計畫。

かねない

接續ます形去ます + かねない

意思很可能造成壞結果

このままでは事故が起こりかねません。
這樣下去可能發生事故。

かねる

接續ます形去ます + かねる

意思難以～

その質問には答えかねます。
那個問題難以回答。

ものだ

接續普通形 + ものだ

意思本來就是；感嘆

時間がたつのは早いものです。
時間過得真快。

ものではない

接續辞書形 + ものではない

意思不應該～

人を見た目で判断するものではありません。
不應以外表判斷人。

次第

接續ます形去ます + 次第

意思一～就～

結果が分かり次第連絡します。
一知道結果就聯絡。

次第で

接續名詞 + 次第で

意思取決於～

努力次第で結果は変わります。
結果取決於努力。

にすぎない

接續普通形 + にすぎない

意思只不過～

これは一例にすぎません。
這只不過是一個例子。

にほかならない

接續名詞 + にほかならない

意思正是～

成功は努力の結果にほかなりません。
成功正是努力的結果。

上で

接續辞書形／た形 + 上で

意思在～之後／方面

内容を確認した上で署名してください。
確認內容後再簽名。
