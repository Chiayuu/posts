N4 文法

欄位：句型／接續／中文意思／日文例句／中文翻譯。

と思います

接續普通形 + と思います

意思我想、我認為

明日は雨が降ると思います。
我覺得明天會下雨。

と言います

接續普通形 + と言います

意思說～

彼は行けないと言いました。
他說不能去。

つもりです

接續辞書形／ない形 + つもり

意思打算～

来年日本へ行くつもりです。
明年打算去日本。

予定です

接續名詞の／辞書形 + 予定

意思預定～

来週出張する予定です。
下週預定出差。

ようと思っています

接續意向形 + と思っています

意思正打算～

日本語の試験を受けようと思っています。
正打算考日文檢定。

ているところです

接續動詞て形 + いるところ

意思正在～

今メールを書いているところです。
現在正在寫信。

たところです

接續動詞た形 + ところ

意思剛剛～

今帰ったところです。
剛回來。

ばかりです

接續動詞た形 + ばかり

意思剛～

日本に来たばかりです。
剛來日本。

ながら

接續ます形去ます + ながら

意思一邊～一邊～

音楽を聞きながら勉強します。
邊聽音樂邊讀書。

ので

接續普通形 + ので

意思因為～所以；較客氣

熱があるので、休みます。
因為發燒，所以休息。

のに

接續普通形 + のに

意思明明～卻～

安いのに、おいしいです。
明明便宜卻很好吃。

てしまう

接續動詞て形 + しまう

意思做完；不小心～

財布を忘れてしまいました。
不小心忘了錢包。

そうです・樣態

接續ます形／い去い／な + そう

意思看起來～

このケーキはおいしそうです。
這蛋糕看起來很好吃。

そうです・傳聞

接續普通形 + そうです

意思聽說～

明日は寒いそうです。
聽說明天很冷。

ようです

接續普通形 + ようです

意思好像～

彼は疲れているようです。
他好像很累。

みたいです

接續普通形 + みたいです

意思像是、好像

この店は有名みたいです。
這家店好像很有名。

なら

接續名詞／普通形 + なら

意思如果是～的話

日本語なら少し話せます。
如果是日文的話會一點。

ば

接續条件形

意思如果～

時間があれば、行きます。
有時間的話會去。

たら

接續た形 + ら

意思如果～；之後～

駅に着いたら、電話してください。
到車站後請打電話。

ても

接續て形 + も

意思即使～也～

高くても買います。
即使貴也會買。

かもしれません

接續普通形 + かもしれません

意思也許～

明日は休みかもしれません。
明天可能休假。

はずです

接續普通形 + はず

意思照理說應該～

彼はもう着いたはずです。
他照理說已經到了。

なければなりません

接續ない形去ない + なければなりません

意思必須～

薬を飲まなければなりません。
必須吃藥。

なくてもいいです

接續ない形去ない + なくてもいい

意思不必～

今日は来なくてもいいです。
今天不用來也可以。

ことにします

接續辞書形／ない形 + ことにします

意思決定～

毎日走ることにしました。
決定每天跑步。

ことになります

接續辞書形／ない形 + ことになります

意思規定／結果變成～

来月転勤することになりました。
下個月要調職了。

てあげる

接續動詞て形 + あげる

意思為別人做～

友達に写真を撮ってあげました。
幫朋友拍了照。

てもらう

接續動詞て形 + もらう

意思請別人做～

先生に説明してもらいました。
請老師說明了。

てくれる

接續動詞て形 + くれる

意思別人為我做～

母が弁当を作ってくれました。
媽媽幫我做便當。

使役

接續させる

意思讓～做

部長は私に資料を作らせました。
主管讓我做資料。
