N1 文法

欄位：句型／接續／中文意思／日文例句／中文翻譯。

いかん

接續名詞 + いかん

意思取決於～

結果はいかんによっては計画を見直します。
取決於結果，可能重新檢討計畫。

いかんにかかわらず

接續名詞 + いかんにかかわらず

意思不論～

理由のいかんにかかわらず欠席は認めません。
不論理由如何都不承認缺席。

を皮切りに

接續名詞 + を皮切りに

意思以～為開端

東京を皮切りに全国で公演します。
以東京為開端在全國演出。

をもって

接續名詞 + をもって

意思以～；到～為止

本日をもって受付を終了します。
到今天為止結束受理。

をよそに

接續名詞 + をよそに

意思不顧～

周囲の心配をよそに彼は挑戦を続けました。
不顧周圍擔心，他持續挑戰。

をものともせず

接續名詞 + をものともせず

意思不把～當回事

困難をものともせず前進しました。
不畏困難前進。

に至るまで

接續名詞 + に至るまで

意思甚至到～

細部に至るまで丁寧に作られています。
連細節都做得很仔細。

に至って

接續名詞／辞書形 + に至って

意思直到～才～

問題が深刻化するに至って対策が取られました。
直到問題嚴重化才採取對策。

に堪えない

接續辞書形／名詞 + に堪えない

意思不忍～；非常～

この映画は見るに堪えない場面があります。
這電影有令人不忍看的場面。

に足る

接續辞書形／名詞 + に足る

意思值得～

信頼するに足る人物です。
是值得信賴的人。

べからず

接續辞書形 + べからず

意思禁止～

関係者以外入るべからず。
非相關人員禁止進入。

まじき

接續名詞 + にあるまじき

意思不應有的～

専門家にあるまじきミスです。
這是專家不該犯的錯。

ずにはおかない

接續ない形去ない + ずにはおかない

意思一定會使～

この話は人々を感動させずにはおきません。
這故事一定會令人感動。

ないではすまない

接續ない形 + ではすまない

意思不～不行

謝らないではすまない状況です。
這情況不道歉不行。

ただではおかない

接續普通形 + ただではおかない

意思不會輕易放過

約束を破ったらただではおきません。
違約的話不會輕易放過。

ともなく

接續辞書形 + ともなく

意思無意識地～

見るともなく窓の外を見ていました。
無意識地看著窗外。

ともあろう

接續名詞 + ともあろう

意思身為～竟然

教師ともあろう人がそんなことを言うとは。
身為老師竟然說那種話。

ながらに

接續名詞 + ながらに

意思保持～狀態

昔ながらの方法で作られています。
以傳統方法製作。

なり

接續辞書形 + なり

意思一～馬上～

彼は部屋に入るなり座り込みました。
他一進房間就坐下。

そばから

接續辞書形／た形 + そばから

意思剛～就～

覚えたそばから忘れてしまいます。
剛記住就忘。

が最後

接續た形 + が最後

意思一旦～就完了

信用を失ったが最後、取り戻すのは難しい。
一旦失去信用就很難挽回。

ものを

接續普通形 + ものを

意思要是～就好了；卻

早く相談してくれればよかったものを。
早點跟我商量就好了。

とあって

接續普通形 + とあって

意思因為～特殊情況

連休とあって駅は混雑しています。
因為連假，車站很擁擠。

とあれば

接續普通形 + とあれば

意思如果是～的話

必要とあれば協力します。
必要的話會協助。

ではあるまいし

接續名詞 + ではあるまいし

意思又不是～

子どもではあるまいし、自分でできます。
又不是小孩，可以自己做。
