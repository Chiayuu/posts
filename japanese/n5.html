N5 文法

欄位：句型／接續／中文意思／日文例句／中文翻譯。

です

接續名詞／な形容詞 + です

意思是、為；禮貌句尾

わたしは学生です。
我是學生。

ます

接續動詞ます形

意思禮貌動詞句尾

毎日日本語を勉強します。
每天學日文。

は

接續A は B です

意思提示主題

これは本です。
這是書。

が

接續名詞 + が + あります／います

意思主語、存在

机の上に本があります。
桌上有書。

を

接續名詞 + を + 動詞

意思受詞標記

水を飲みます。
喝水。

に

接續時間／地點 + に

意思時間點、目的地、存在地

七時に起きます。
七點起床。

で

接續場所／工具 + で

意思動作地點、方法工具

図書館で勉強します。
在圖書館讀書。

へ

接續場所 + へ行く

意思方向

日本へ行きます。
去日本。

と

接續名詞 + と + 名詞／人と

意思和、一起

友達と映画を見ます。
和朋友看電影。

も

接續名詞 + も

意思也

私も台湾人です。
我也是台灣人。

の

接續名詞 + の + 名詞

意思的、所屬

これは私のかばんです。
這是我的包包。

から／まで

接續A から B まで

意思從～到～

九時から五時まで働きます。
從九點工作到五點。

じゃありません

接續名詞／な形容詞 + じゃありません

意思不是

今日は休みじゃありません。
今天不是休假。

でした／じゃありませんでした

接續名詞／な形容詞 + 過去

意思過去式

昨日は雨でした。
昨天是雨天。

い形容詞

接續い形容詞 + です

意思描述性質

このケーキはおいしいです。
這個蛋糕很好吃。

な形容詞

接續な形容詞 + な + 名詞

意思描述性質

静かな町です。
安靜的城鎮。

あります／います

接續物があります／人・動物がいます

意思存在；無生命／有生命

公園に犬がいます。
公園裡有狗。

ませんか

接續動詞ませんか

意思邀請

一緒に昼ご飯を食べませんか。
要不要一起吃午餐？

ましょう

接續動詞ましょう

意思一起做吧

休みましょう。
休息吧。

たい

接續動詞ます形去ます + たい

意思想要做

日本へ行きたいです。
想去日本。

てください

接續動詞て形 + ください

意思請做～

ここに名前を書いてください。
請在這裡寫名字。

てもいいです

接續動詞て形 + もいいです

意思可以做～

写真を撮ってもいいです。
可以拍照。

てはいけません

接續動詞て形 + はいけません

意思不可以做～

ここでタバコを吸ってはいけません。
這裡不能抽菸。

ないでください

接續動詞ない形 + でください

意思請不要～

心配しないでください。
請不要擔心。

ことができます

接續動詞辞書形 + ことができます

意思能夠做～

少し日本語を話すことができます。
會說一點日文。

たことがあります

接續動詞た形 + ことがあります

意思曾經～

京都へ行ったことがあります。
曾經去過京都。

たり〜たりします

接續動詞た形 + り、動詞た形 + りします

意思列舉動作

週末は掃除したり、買い物したりします。
週末會打掃、買東西等等。

ほしい

接續名詞 + がほしい

意思想要東西

新しいパソコンがほしいです。
想要新電腦。

より／のほうが

接續A より B のほうが〜

意思比較

電車より車のほうが便利です。
車比電車方便。

いちばん

接續範圍 + で + いちばん

意思最～

季節で夏がいちばん好きです。
季節裡最喜歡夏天。
