N3 文法

欄位：句型／接續／中文意思／日文例句／中文翻譯。

ために

接續辞書形／名詞の + ために

意思為了～

合格するために毎日勉強しています。
為了合格每天讀書。

ように

接續辞書形／ない形 + ように

意思為了能～

忘れないようにメモします。
為了不要忘記而做筆記。

ようになる

接續辞書形 + ようになる

意思變得會～

漢字が読めるようになりました。
變得會讀漢字了。

ことになっている

接續辞書形 + ことになっている

意思規定、安排

ここでは靴を脱ぐことになっています。
這裡規定要脫鞋。

わけではない

接續普通形 + わけではない

意思並不是～

嫌いなわけではありません。
並不是討厭。

わけがない

接續普通形 + わけがない

意思不可能～

彼が知らないわけがありません。
他不可能不知道。

わけにはいかない

接續辞書形 + わけにはいかない

意思不能那樣做

約束があるので、帰るわけにはいきません。
因為有約，不能回去。

はずがない

接續普通形 + はずがない

意思不可能～

そんなに安いはずがない。
不可能那麼便宜。

べき

接續辞書形 + べき

意思應該～

約束は守るべきです。
約定應該遵守。

たびに

接續辞書形／名詞の + たびに

意思每當～

この歌を聞くたびに、台湾を思い出します。
每次聽到這首歌就想起台灣。

最中に

接續名詞の／ている + 最中に

意思正在～時

会議の最中に電話が鳴りました。
會議中電話響了。

うちに

接續普通形 + うちに

意思趁～期間

若いうちにいろいろ挑戦したいです。
想趁年輕多挑戰。

間／間に

接續普通形 + 間に

意思在～期間

日本にいる間に温泉へ行きたいです。
在日本期間想去溫泉。

ところが

接續句子。ところが、

意思然而

急いで行きました。ところが、店は閉まっていました。
急忙去了，結果店關了。

つまり

接續句子。つまり、

意思也就是說

彼は来ません。つまり、私たちだけで行きます。
他不來，也就是說只有我們去。

によって

接續名詞 + によって

意思根據；因～而不同

国によって習慣が違います。
國家不同習慣也不同。

について

接續名詞 + について

意思關於～

日本の文化について調べています。
正在查日本文化。

に対して

接續名詞 + に対して

意思對於～

この意見に対して反対です。
對這個意見反對。

として

接續名詞 + として

意思作為～

エンジニアとして働いています。
作為工程師工作。

によると

接續名詞 + によると

意思根據～

天気予報によると、明日は雨です。
根據天氣預報明天會下雨。

向け／向き

接續名詞 + 向け／向き

意思面向／適合

これは初心者向けの本です。
這是給初學者的書。

ほど

接續普通形 + ほど

意思程度；越～越～

考えるほど難しくなります。
越想越難。

くらい／ぐらい

接續普通形 + くらい

意思程度

泣きたいくらい悔しかったです。
懊悔到想哭。

ばかりでなく

接續A ばかりでなく B も

意思不只～也～

彼は英語ばかりでなく日本語も話せます。
他不只會英文也會日文。

だけでなく

接續A だけでなく B も

意思不只～也～

この店は安いだけでなく、おいしいです。
這家店不只便宜也好吃。

に違いない

接續普通形 + に違いない

意思一定～

彼は本当のことを知っているに違いない。
他一定知道真相。

恐れがある

接續辞書形／名詞の + 恐れがある

意思有可能發生壞事

台風で電車が止まる恐れがあります。
颱風可能導致電車停駛。

ことはない

接續辞書形 + ことはない

意思不需要～

そんなに心配することはありません。
不用那麼擔心。

しかない

接續辞書形 + しかない

意思只好～

もうやるしかありません。
只能做了。

ついでに

接續辞書形／名詞の + ついでに

意思順便～

買い物のついでに郵便局へ行きます。
買東西順便去郵局。
